# Emergency-Aware School Bus Tracking System

## Domain
Emergency Systems

## Description
A real-time school bus tracking and emergency alert system designed to ensure student safety and enable fast emergency response.

## Features
- Live bus tracking
- Emergency (SOS) alerts
- Parent & admin notifications
- Incident reporting
- Cloud & DevOps ready architecture

## Tech Stack
- Java, Spring Boot
- MySQL
- Git & GitHub
- Jenkins
- Docker, Kubernetes
- AWS
